# Virtual Lost and Found System

A web-based Lost and Found System developed using PHP and CodeIgniter 4 framework.  
This system allows users to report lost or found items and enables administrators to manage and approve listings efficiently.

---

## 🚀 Features

- User Registration & Login
- Admin Authentication
- Post Lost Items
- Post Found Items
- Recent Listings Display
- Admin Dashboard
- Approve / Reject Submissions
- Manage Users (Block / Suspend / Reset Password)
- Announcements Management

---

## 🛠️ Technologies Used

- PHP
- CodeIgniter 4
- MySQL
- Bootstrap 5
- HTML & CSS
- MVC Architecture

---

## 🔐 Security Features

- Authentication & Authorization
- Role-Based Access Control
- Form Validation
- Secure Session Handling

---

## ⚙️ Installation Steps

1. Clone the repository
   git clone https://github.com/tanishamanani/virtual-lost-and-found-system.git
   
2. Move into project directory
   cd project-folder

3. Install dependencies
   composer install
   
4. Configure `.env` file
- Set baseURL
- Add database credentials

5. Run migrations (if used)
   php spark migrate

6. Start development server
   php spark serve


---

## 📌 Project Purpose

This project was developed as a college major project to simplify the process of reporting and managing lost and found items within an institution.
